### <img src="assets/icons/terminal.svg" width="22" height="22" align="center" /> Self-hosted — Versión web

Aloja tu propia instancia con autenticación y base de datos en la nube.

**Necesitas:** [pnpm](https://pnpm.io/installation) · [Neon PostgreSQL](https://neon.tech) (gratuito)

```bash
# 1. Clonar e instalar dependencias
git clone https://github.com/SwonDev/Stacklume.git
cd Stacklume
pnpm install

# 2. Configurar entorno
cp .env.example .env.local
# → Edita .env.local con tu DATABASE_URL y credenciales

# 3. Crear las tablas en la base de datos
pnpm db:push

# 4. Iniciar servidor de desarrollo
pnpm dev
```

Abre [http://localhost:3000](http://localhost:3000). Consulta [`.env.example`](.env.example) para ver todas las variables disponibles.
