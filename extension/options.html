Configuración de la extensión

Instancia de Stacklume

URL donde corre la app de escritorio Stacklume (Tauri). En producción usa el puerto 7879; en desarrollo, el 7878.

Presets:

Token de API (MCP)

Necesario para guardar enlaces directamente sin abrir Stacklume. Encuéntralo en Ajustes → MCP dentro de la app de escritorio.

Sin token, la extensión abrirá Stacklume en una nueva pestaña pre-rellenada.

Comportamiento

Requiere token de API configurado.

Atajo de teclado

El atajo predeterminado es Alt+S. Puedes cambiarlo desde chrome://extensions/shortcuts (Chrome/Edge) o about:addons → Administrar atajos (Firefox).
