Instancia de Stacklume

URL donde corre la app de escritorio Stacklume (Tauri). En producción usa el puerto 7879; en desarrollo, el 7878.

Presets:

Token de API (MCP)

Necesario para guardar enlaces directamente sin abrir Stacklume. Encuéntralo en Ajustes → MCP dentro de la app de escritorio.

Sin token, la extensión abrirá Stacklume en una nueva pestaña pre-rellenada.

Comportamiento

Requiere token de API configurado.

Atajo de teclado

El atajo predeterminado es Alt+S. Puedes cambiarlo desde chrome://extensions/shortcuts (Chrome/Edge) o about:addons → Administrar atajos (Firefox).

Gestionar atajos del navegador →

Datos y respaldo

Exporta o importa la configuración de la extensión (URL, token, preferencias) para restaurarla en otro navegador o perfil.

Privacidad

Tus datos se envían únicamente a tu instancia local de Stacklume.
La extensión no recopila telemetría ni envía datos a servidores de terceros.
Solo accede al título, URL y metadatos de la pestaña activa cuando la invocas.
El token de API se almacena en chrome.storage.sync (cifrado por el navegador).
