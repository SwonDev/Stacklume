## Full Description (for the store)

Stacklume is the companion extension for the Stacklume desktop app — an intelligent bookmark manager with local AI.

**Save any link with one click:**
- Automatically captures title, description, favicon and page image
- Detects platforms (YouTube, GitHub, Steam, Twitter/X, Reddit, npm, Figma...)
- Detects commands (npm install, pip install, cargo add, brew install...)
- Context menu: save pages, links and images with right-click

**Organize your way:**
- Categories and tags with colors
- Reading status: Inbox / Reading / Done
- Personal notes and date reminders
- Favorites with one click

**Productivity:**
- Save all open tabs at once
- Smart duplicate detection (normalizes URLs, strips UTM params)
- Keyboard shortcut Alt+S for quick save

**Total privacy:**
- Your data is stored locally on your computer
- No telemetry, no tracking, no external servers
- Works with the Stacklume desktop app (free and open source)

**Requirements:**
- Stacklume desktop installed on your computer (available on GitHub)
- API token generated from Settings > MCP in the app

**Open source:** https://github.com/SwonDev/Stacklume
